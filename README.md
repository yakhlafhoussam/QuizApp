# 🎯 Quiz App

A modern and interactive **Quiz / QCM application** built with **Vue 3 + TypeScript + Vite**.

This project was created as part of a frontend challenge to test technical knowledge through a **progressive multi-level quiz system** with a **global timer**, **real-time score tracking**, and a clean user experience.

---

## 🚀 Features

### ✅ Core Features
- Home page with quiz introduction
- 3 progressive quiz levels
- 5 questions per level
- 4 multiple-choice answers per question
- Real-time score system
- Global timer (**5 minutes**)
- Automatic level validation
- Win / Lose / Timeout result screens

### 🎮 Game Rules
- **20 points** per correct answer
- **Level 1** → minimum **40/100**
- **Level 2** → minimum **60/100**
- **Level 3** → minimum **80/100**

### 💡 Bonus Features
- Progress bar
- Correct / wrong answer feedback
- Auto-next question transition
- Sound effects:
  - Correct answer
  - Wrong answer
  - Win
  - Game Over
- State persistence with `localStorage`
- Responsive UI
- JSON-based mock data structure ready for API integration
